<h1>keruoji</h1>


可萝仔插件是一个Yunzai-Bot的扩展插件，给bot提供一些有的没得功能希望能有所帮助

</div>


#### 安装教程

本插件只适配了 Yunzai-Bot v3 安装前请确认版本
将keruoji下载下来放在Yunzai-Bot/plugins目录下，重启Yunzai-Bot后可正常使用。

 -- 使用Gitee安装
```
git clone https://gitee.com/levebo/keruoji.git  ./plugins/hs-qiqi-plugin
```
#### 使用说明

<details>
